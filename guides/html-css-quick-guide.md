# HTML + CSS Quick Guide

## Core Idea
HTML builds structure. CSS applies style.

## Must-Know Words
- tag: HTML element marker
- attribute: extra info in tag
- selector: CSS way to pick elements
- property/value: CSS rule pair

## Logic Pattern
1. Write element in HTML
2. Select it in CSS
3. Apply style rules

## Tiny Example
```html
<h1 class="title">Welcome</h1>
```

```css
.title { color: teal; }
```

# Coding Glossary for Kids

- algorithm: step-by-step way to solve a problem
- bug: error in code
- debug: finding and fixing bugs
- compiler: tool that converts C/C++ code into machine code
- interpreter: tool that runs code line by line (common in Python)
- logic: the thinking rules behind code
- syntax: grammar rules of a programming language
- runtime: when a program is currently running
- output: result shown by the program
- input: data given to a program

# JavaScript Quick Guide

## Core Idea
JavaScript makes websites interactive and smart.

## Must-Know Words
- let/const: variable declarations
- function: reusable task
- event: action like click or key press
- DOM: web page elements you can control

## Logic Pattern
1. Select element
2. Listen for event
3. Run function on event

## Tiny Example
```js
const btn = document.getElementById("btn");
btn.addEventListener("click", () => {
  console.log("Clicked");
});
```

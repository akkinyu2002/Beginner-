# C++ Quick Guide

## Core Idea
C++ is C plus object-oriented features (classes and objects).

## Must-Know Words
- class: blueprint
- object: real item created from class
- method: function inside a class
- cout: print text

## Logic Pattern
1. Define class
2. Create object
3. Call methods to do work

## Tiny Example
```cpp
class Box {
public:
  void show() { cout << "Hi"; }
};
```

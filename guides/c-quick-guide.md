# C Quick Guide

## Core Idea
C is a fast and powerful language where you learn basic logic deeply.

## Must-Know Words
- variable: storage box for data
- int: whole number type
- float: decimal number type
- scanf: take input
- printf: show output

## Logic Pattern
1. Start at `main()`
2. Take input
3. Process with formulas or conditions
4. Print output

## Tiny Example
```c
int a = 5;
int b = 3;
printf("%d", a + b);
```

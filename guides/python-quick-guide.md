# Python Quick Guide

## Core Idea
Python is clean and easy to read, great for beginners.

## Must-Know Words
- variable: name that stores data
- if/else: decision making
- for loop: repeating steps
- function: reusable code block

## Logic Pattern
1. Input data
2. Use conditions/loops
3. Print final result

## Tiny Example
```python
name = "Asha"
if name == "Asha":
    print("Hello")
```

Print and Variables

age = 10

If Condition

marks = 85

if marks >= 50:
    print("Pass")
else:
    print("Try Again")

Logic: If condition is true, first block runs. Otherwise else block runs.

Loop

for i in range(1, 6):
    print(i)

This loop repeats from 1 to 5. Great for repeated tasks.

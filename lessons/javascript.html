Variable

A variable stores data in a named box.

let studentName = "Aman";

Function

A function is a task machine. You call it when you need that task.

function greet(name) {
  return "Hello " + name;
}

console.log(greet("Riya"));

Event Logic

When user clicks a button, code runs. That is event-based logic.

const btn = document.getElementById("myBtn");
btn.addEventListener("click", () => {
  alert("You clicked me!");
});

What is HTML?

HTML uses tags to build page parts like heading, paragraph, button, and image.

<h1>My First Page</h1>
<p>I am learning code!</p>

What is CSS?

CSS changes how HTML looks: color, size, spacing, and layout.

h1 {
  color: #0a5f87;
}

p {
  font-size: 20px;
}

Logic Behind Design

Step 1: Add content in HTML.

Step 2: Select that content in CSS.

Step 3: Give style rules. Browser applies those rules to matching elements.

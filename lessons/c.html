Hello Program

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

main() is where the program starts. printf prints text.

Input + Output

#include <stdio.h>

int main() {
    int age;
    scanf("%d", &age);
    printf("Age is %d\n", age);
    return 0;
}

scanf takes user input. %d means integer value.

Loop Logic

for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

Start at 1, keep going while i is less than or equal to 5, increase i each round.

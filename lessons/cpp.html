Class + Object

#include <iostream>
using namespace std;

class Robot {
public:
    void speak() {
        cout << "I am a learning robot" << endl;
    }
};

int main() {
    Robot r1;
    r1.speak();
}

Class is a blueprint. Object is a real thing made from that blueprint.
